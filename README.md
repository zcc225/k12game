# k12game

🎮 **幼小衔接学习游戏平台** - 边玩边学，快乐成长！

## 游戏模块

| 模块 | 图标 | 描述 | 学习内容 |
|------|------|------|----------|
| 识字乐园 | 📚 | 认识汉字朋友 | 基础汉字认读、拼音、词语 |
| 数学天地 | 🔢 | 学习加减运算 | 10以内加减法、可视化计数 |
| 认知世界 | 🎨 | 认识颜色形状 | 颜色分类、形状识别 |
| 拼音王国 | 🎵 | 学习声母韵母 | 声母认读、拼音发音 |
| 比较大小 | ⚖️ | 认识数字大小 | 数量比较、大于小于等于 |
| 找规律 | 🔄 | 发现图形规律 | 序列推理、模式识别 |
| **拼图乐园** | 🧩 | **拼图学知识** | **数字/字母/图形/汉字拼图** |

## 🆕 拼图乐园功能

### 四大拼图类型

1. **🔢 数字拼图** - 学习数字1-8、奇偶数、序数词
2. **🔤 字母拼图** - 学习英文字母A-H、大小写、元音字母
3. **🔷 图形拼图** - 学习颜色、形状、水果、花朵
4. **汉字拼图** - 学习数字汉字、方位词、四季汉字

### 三级难度

- **简单 2×2** - 4块拼图，适合初学者
- **中等 3×3** - 9块拼图，有一定挑战
- **困难 4×4** - 16块拼图，高级玩家

### 知识学习融入

每完成一个拼图，都会展示相关的知识点，例如：
- 数字顺口溜："1像铅笔细又长，2像小鸭水上漂..."
- 英文字母形状记忆："A像尖塔，B像耳朵..."
- 汉字笔画规则："从上到下、从左到右"
